## WordPress.org & WordPress.com

- .com is fully hosted server, which we sign up for build blog or website without having to manage the underline technology ourself.
- It is like renting an apartment in a complex,You don't have to worry about the functionality but you can set your apartment as you want. But you can renovate the apartment without permission.
- It does not allow as all users use same platforms and access to all can cause security issue.

- .org is home for free wordPress software, that we can install on our webserver.
- Moving to self hosted site, looks like buying your own house.
- You are responsible for performing rotine updates, backing up your own content 
- You are bounded by your own rules.
- Self hosted Version, full control

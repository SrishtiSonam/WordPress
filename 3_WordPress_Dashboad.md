                      : Option to see admin bar at the top of your website.

                      : Other Usage like password change, log out from other, etc

- Settings Menu : General : Bunch of global settings for your account.
                          : Ste title, tagline(based on theme usage), 
                          : Never change your WordPress Address URL.  
                          : Site Address URL - Different homepage from where your wordpress installations located. {Example - Wordpress is attached to a site. (Part of it) }
                          : Administration Email Address - Mail for official wordPress mails.
                          : Membership (People creating there own account) - For community websites.

- Settings Menu : Writing : Most likely not going to change anything on this website.
                          : Default Post Category : Every Post have atleast one category associated to it.
                          : Similarly Default Post Format
                          : Post via email : Adding a post by email content, with login name, password, default category, update server.  { Not recommended }
                          : Update server : Automatically send updates to search engines.

- Settings Menu : Reading : Blog Pages shown at most { According to grid format design }
                          : Syndication feeds show the most recent : For RSS feed
                          : For each post in a feed, includes : For RSS feed
                          : Search engine visibility 

- Settings Menu : Discussion : How our website manages comments
                            : Default Post Settings : Controls whether posts allow comments, send notifications, and receive pingbacks by default.
                            : Other Comment Settings : Defines who can comment, enables threaded replies, and sets auto-closing for old posts.
                            : Comment Pagination : Manages how comments are split into pages and displayed for better readability.
                            : Email Notifications : Notifies the admin when new comments are posted or need moderation.
                            : Before a Comment Appears : Controls whether comments need approval before being published.
                            : Comment Moderation : Filters spam by holding suspicious comments for review.
                            : Disallowed Comment Keys : Automatically deletes comments containing banned words or patterns.
                            : Avatars : Manages display and type of user profile images in comments.

- Settings Menu : Media : Image Sizes - Defines default dimensions for images uploaded to the Media Library.
                            :: Thumbnail Size - Sets small preview image size used in galleries and listings. {Option of Croping}
                            :: Medium Size - Controls size for moderate display in posts and pages. {Maintain the aspect ratio}
                            :: Large Size - Determines size for full-width or featured image display. {Maintain the aspect ratio}
                            The larger side is set according to these sizes and then according to aspect ratio other side is set.
                            :: A full size option will also be there to show image with actual size.
                        : Uploading Files - Organizes uploaded media into month- and year-based folders for better file management.

- Settings Menu : Permalink : The site important thing to do with website setting is configuring this.
                            : It determines what your post and page URLs will look like.
                            : Initially it will not be search engine friendly. { We want URL that is readable to humans and also have ability to tell search engines what the page is about. }
                            : You have to choose or prefer any custom structure.
                            : It is prefered to not change this once choosen, as this is used to for whole website system.
                            : An option to enter custom structure for your category and URLs. { Like instead of category you can have topics etc }

- Settings Menu : Privary : For privary policies pages. 
                          : It analyze current configurations, and create customise private policies with major section. ( which is editable  -  specially remove suggested text )



// WP offers you the ability to add additional users to you website for variety of purposes.
    These individual users can be offered specific privileges.

Managing users for your website :-

- Users Menu : With familiar designs.
             : Add new option. Here selected role define which privileges will be provided to this user.
                - Subscriber : Read content & manage their profile.
                - Contributor : Create and Edit their own post, Submit post for review by an editor / admin.
                - Author : Create, Edit and Publish their own posts. And upload media attachments. { Unrestricted Publishing Rights }
                - Editors : Create, Edit, Publish and Delete any posts or pages, including those created by others. And upload media attachments. And manage categories. And moderate comments. { Only manage the content not the global settings of the website. }   
                - Administrator : Site owner. Editor roles + Customise Themes + Import & Export Content + Add, Manage and Delete users including other administrators + Install, Active & Delete themes and plugins + Install core, plugins and themes updates + Edit Core. 
             : When you create the users, you can edit their profile as you can of your own.
